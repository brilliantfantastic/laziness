laziness
========

## DESCRIPTION

A [Slack](http://slack.com) [API](http://api.slack.com) wrapper written in Ruby.

## INSTALLATION

Add this line to your Gemfile:

```
  gem 'laziness'
```

And then execute:

```
  bundle install
```

Or install it yourself:

```
  gem install laziness
```
